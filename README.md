# 🎓 Aprendendo IHC

Um Recurso Educacional Aberto (REA) para o ensino de Interação Humano-Computador (IHC), apresentado em um site moderno, intuitivo e interativo.

---

## 📄 Sobre o Projeto

O projeto **Aprendendo IHC** foi desenvolvido como um Recurso Educacional Aberto (REA) com o objetivo de auxiliar estudantes no aprendizado dos principais conceitos de Interação Humano-Computador.

A plataforma reúne conteúdos organizados em videoaulas, slides e listas de exercícios, proporcionando uma experiência de aprendizagem acessível, prática e interativa. Os materiais abordam temas fundamentais da área, como Fundamentos de IHC, Usabilidade, Design de Interfaces, Experiência do Usuário (UX), Acessibilidade e Heurísticas.

---

## ✨ Tecnologias Utilizadas

Este projeto foi construído utilizando as seguintes tecnologias:

* **HTML5:** Estrutura semântica e organização do conteúdo.
* **CSS3:** Estilização, responsividade e animações.
* **JavaScript:** Interatividade da interface e funcionamento dos modais.
* **Font Awesome:** Biblioteca de ícones utilizada no projeto.

## ▶️ Como Executar o Projeto

Este é um projeto web estático. Para executá-lo localmente, siga os passos abaixo:

1. Clone o repositório:

```bash
git clone https://github.com/seu-usuario/aprendendo-ihc.git
```

2. Acesse a pasta do projeto:

```bash
cd aprendendo-ihc
```

3. Abra o arquivo `index.html` em seu navegador.
